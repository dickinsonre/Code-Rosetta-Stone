```

  │                                                       │
  │  Question 3 of 5:                                     │
  │                                                       │
  │  In the friction slope computation, why is the        │
  │  sign of Sf reversed when Q < 0?                     │
  │                                                       │
  │  ○ A) Because negative flow means uphill flow        │
  │  ● B) Because friction always opposes flow direction  │  ← CORRECT
  │  ○ C) Because Manning's n is negative for reverse flow│
  │  ○ D) Because the hydraulic radius becomes negative  │
  │                                                       │
  │  ✅ CORRECT!                                         │
  │  Friction slope magnitude is always positive          │
  │  (Sf = [n·Q/(A·R^(2/3))]²), but its sign must match │
  │  the flow direction to correctly oppose flow in the   │
```
