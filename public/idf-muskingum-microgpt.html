IDF Curves — Discover the Power Law

Every storm drain design starts with an IDF curve: intensity vs duration for different return periods. The underlying function is i = a·Tm/(t+b)c where T = return period, t = duration, and a,b,c,m are regional constants. The network learns these power-law relationships from data alone — no formula given.

Muskingum Routing — Attenuation & Translation

The Muskingum method routes an inflow hydrograph through a channel reach, producing an outflow that is attenuated (lower peak) and translated (delayed). K controls the travel time, X controls the wedge storage (0 = maximum attenuation, 0.5 = pure translation). The network learns the routing coefficients C₀, C₁, C₂ from scalar parameters.
